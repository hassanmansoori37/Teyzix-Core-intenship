## Overview

TaskFlow Pro is a modern SaaS project management platform featuring a complete landing page, customer onboarding portal, and dashboard preview.

---

## Pages

| Route | Description |
|---|---|
| `/` | Landing page (Hero, Features, Pricing, FAQ, Contact) |
| `/login` | Login page with form validation |
| `/signup` | Signup page with password strength indicator |
| `/onboarding` | 3-step onboarding portal with progress indicator |
| `/dashboard` | Dashboard preview with KPIs and activity feed |

---

## Tech Stack

| Layer | Technology |
|---|---|
| UI | React 18 + TypeScript |
| Styling | Tailwind CSS |
| Routing | React Router v6 |
| Theme | Context API (dark/light) |

---

## Features

- ✅ Hero section with CTA buttons

## Getting Started

```bash
npm install --legacy-peer-deps
npm start
```

Open `http://localhost:3000`

---

## Deployment

```bash
npm run build
```

Deploy `/build` folder to Netlify or Vercel.
